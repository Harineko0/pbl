## Git
```shell
# リポジトリをクローン
git clone https://github.com/Harineko0/pbl

# エディアでプログラミングを編集

git add <ファイルを追加した場合,追加したファイルのパス>
git commit -am "変更内容を書いたメッセージ"
git push origin main
```

2. 拡張機能 -> App Script
3. 左側の歯車アイコンをクリック
4. ページの中部に スクリプトID がある
